# 🚀 太空战机：无限火力

一款基于 HTML5 Canvas 的网页射击游戏，支持桌面和移动端。

## 🎮 在线试玩

**[点击开始游戏](https://dongweiq.github.io/space-fighter/)**

## ✨ 游戏特色

- 🎯 经典飞机大战玩法
- 👾 多种敌机和 Boss 战
- ⚡ 丰富的武器升级系统
- 💎 资源收集与能量管理
- 📱 完美支持手机触屏操作
- 🎨 炫酷的视觉特效

## 🕹️ 操作方式

| 平台 | 操作 |
|------|------|
| 桌面 | 鼠标/键盘控制移动，自动射击 |
| 手机 | 触屏滑动控制战机 |

## 🛠️ 技术栈

- HTML5 Canvas
- Tailwind CSS
- 原生 JavaScript
